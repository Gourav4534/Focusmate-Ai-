# FocusMate AI - Goal Consistency App

An AI-powered application that helps users stay consistent with their goals by offering emotional support, daily accountability, progress tracking, and personalized nudges to overcome distractions.

## Features

- Goal Setting and Motivation Tracking
- Daily Check-ins and Progress Monitoring
- Task Management and Prioritization
- Emotional Support and AI-powered Nudges
- Progress Visualization and Analytics
- Weekly Reflection and Adjustment Tools

## Tech Stack

- React with TypeScript
- Material-UI for UI components
- Redux Toolkit for state management
- React Router for navigation
- Chart.js for data visualization
- Vite for build tooling

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Project Structure

```
src/

  ├── store/         # Redux store and slices
  ├── types/         # TypeScript type definitions
  ├── utils/         # Utility functions

```
